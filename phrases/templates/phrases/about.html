📚 About the Project

🎯 Project Objective

This project is a Motivational Phrase Generator developed with Django. It allows users to select a specific category (work, study, love, sports, life) and get a random motivational phrase corresponding to that category.

✅ Requirements Met

  • ✅ Web framework: Django
  • ✅ 2 dynamic HTML pages: Main page and information page
  • ✅ Interactivity: Category selection and phrase generation
  • ✅ Local execution: Django development server
  • ✅ Database: SQLite with Django models
